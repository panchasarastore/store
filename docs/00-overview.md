# Panchasara – Project Overview

Last updated: 2026-02-06

## What is Panchasara?
Panchasara is a platform for home bakers, artisans, and small-scale businesses to create and manage their own online stores.

## Target Users
Store Owners (to manage) and Buyers (to purchase).

## Current Stage
Beta Development

- Phase 1: Authentication ✅
- Phase 2: Store Creation ✅
- Phase 3: Storefront & Checkout (Active) 🛠️
- Phase 4: Post-Payment (Notifications & Analytics - Planned)

## Tech Stack
- **Frontend**: Astro (SSR)
- **Backend & Auth**: Supabase
- **Database**: PostgreSQL (via Supabase)
- **Payments**: Cashfree (Integrated via Edge Functions)
- **Email**: Resend (Planned)
